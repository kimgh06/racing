기본 엔진

- 플레이어 오브젝트의 진짜 차같은 조작감 필요 없음, 지금은 단순히 어린이가 차를 가지고 노는 것처럼 조작이 되야 함.
  - 이동은 무조건 프레임이 아닌 큐를 이용해서 처리하도록 할 것
  - 그냥 캐릭터가 머리 까딱 까딱 움직이기만 하면 됨.
  - 디버깅용 리플레이와 고스트 기능 필요
- 세이브 포인트로 정당하게 돌았는 지 필요, z축에 회전에 따른 플레이어 초기화 로직 개선 필요
- 차는 단순한 도형으로만 처리하고, 필요할 경우 텍스쳐를 적용시키도록 처리
  차를 커스텀이 가능하게 하기 위한 데이터 저장 방법.
- 시간 기록이나 위치 데이터를 통해 기록을 남길 용도.
- 멀티 플레이를 위해 어떻게 물리엔진을 이용할 건지.
  위치, 회전 정보를 계속 동기화(csv 구조로 사용할 예정)
- 전광판은 어떻게 매달 것인지.
- 웹 서비스로 멀티 운영이 가능해야 함.
- 넣으면 재밌겠는 기능들
  - 오브젝트를 집어서 멀리 보내는 기능
  - 점프 기능
